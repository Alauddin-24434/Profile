# 🏨 Royal Place - Luxury Hotel & Resort Management System

Royal Place is a premium hotel management platform built with modern technologies to provide a seamless booking experience and efficient resort management.

## 🚀 Tech Stack
- **Frontend:** Next.js 15 (Turbopack), Tailwind CSS 4, Framer Motion.
- **State Management:** Redux Toolkit & Redux Persist.
- **Payments:** Stripe Integration.
- **AI Integration:** OpenAI SDK for guest assistance.
- **Utilities:** Zod, Date-fns, Lucide React.

## ✨ Main Features
- 🛎️ **Luxury Booking:** Intuitive room booking flow with real-time availability.
- 💳 **Secure Payments:** Full Stripe integration for room reservations and services.
- 🤖 **AI Concierge:** AI-powered chatbot to assist guests with inquiries.
- 🎭 **Smooth Animations:** Premium feel using Framer Motion and Tailwind 4.
- 🌐 **Multilingual:** Ready for global guests with internationalization support.
- 📅 **Schedule Management:** Integrated with Calendly for resort activities.

## 📦 Dependencies
- `next`, `tailwindcss`, `@stripe/stripe-js`, `@reduxjs/toolkit`, `framer-motion`, `openai`, `zod`.

## 🛠️ Local Setup Guide

### 1. Clone the repository
```bash
git clone https://github.com/dev-alauddin-bd/royal-palce-frontend.git
cd royal-palce-frontend
```

### 2. Install dependencies
```bash
npm install
```

### 3. Environment Variables
Create a `.env` file in the root:
```env
NEXT_PUBLIC_STRIPE_PUBLISHABLE_KEY=your_key
OPENAI_API_KEY=your_key
```

### 4. Run the project
```bash
npm run dev
```
